About


Flip-UI is a simple application that provides you with multiple UI designs by switching different bootswatch themes Flip-UI runs on a simple nodejs server (backend), while HTML5, CSS3 (custom stylesheet and bootstrap framework) and jquery, powers the frontend..

How It Works


Frontend Logic

The moment a user submits a theme selection, a jquery click event listener is triggered, which the makes a GET request to the server either via XHttp, ajax or fetch function, depending on the submit button clicked by the user. Being asynchronous, ajax and fetch functions waits for the server's response data. Looping through the theme object name and cdn properties (server response data ), using an identifier passed as a callback parameter, if a name matches the value of the intial ui design selected, the respective theme bootswatch cdn is then set as the <link src=""> attribute for the first Link child of the head element using jquery for all the pages.

Backend Logic

All request are served back to the client using node.Js fs and path modules. Once a "/theme" request gets to the server; the server checks within the application directory for a themes.json file, that contains an object of the different themes in the app directory and sends back the data in the file as response for further client-side processing. A simple Switch conditional is used to set the appropraite content-type header for response sent back to the client.
